欢迎使用 Moon小手机！

1. 本应用为纯前端网页应用，所有数据均保存在您的本地浏览器缓存中。请务必定期在设置中导出备份数据，清理缓存或更换设备会导致数据永久丢失！

4. 请遵守相关法律法规，不要使用本应用生成违规内容。

新手教程
Moon小手机 使用指南
1. 核心驱动：配置 API
Moon小手机的所有 AI 交互（聊天、朋友圈、日记等）都需要大语言模型 API 的支持。

操作步骤：
1. 在桌面点击【设置】应用。
2. 找到“核心驱动引擎 (API)”板块。
3. 填入你的 接口地址 (API URL) 和 API 密钥 (API Key)。例如使用 OpenAI 或兼容格式的第三方中转接口。
4. 点击刷新按钮获取可用模型，或手动输入模型名称（如 gpt-4o 或 claude-3-5-sonnet）。
5. 点击“测试连接状态”，如果提示成功，就可以点击右上角“保存”。

💡 提示：你可以将常用的接口保存为“API 预设”，方便以后快速切换。
2. 创建你的第一个联系人
有了 API 后，你需要创建 AI 角色来互动。

操作步骤：
1. 在桌面点击【聊天】应用。
2. 切换到下方第二个 Tab（联系人）。
3. 点击右上角带有“+”号的悬浮按钮，进入添加联系人页面。
4. 设置头像，填写“名字”和“人设”。人设是 AI 扮演角色的核心，请尽量详细描述其性格、身份、说话方式等。
5. 选填“我们之间的经历”和“双向视角”，这能让 AI 更好地理解你们的关系。
6. 点击右上角“保存”。
3. 开始聊天与高级互动
基础聊天：在联系人列表点击角色，进入聊天室。在底部输入框发送消息，AI 会根据人设自动回复。

多媒体与转账：点击输入框左侧的笑脸可发送表情包（需先在设置中添加）；点击右侧的“+”号可发送图片、虚拟图片或发起转账互动。

长按消息：长按任意消息气泡，可以进行回复、收藏、多选转发或删除操作。

聊天设置：点击聊天室顶部的角色名字，进入专属聊天设置。在这里你可以：
- 修改双方头像和气泡颜色。
- 绑定“世界书”以注入世界观背景。
- 开启“旁白模式”，让 AI 在对话中加入动作和神态描写。
- 管理角色的表情包库和专属头像库（AI 会根据情境自动使用）。
4. 朋友圈与生态记忆
朋友圈：在【聊天】应用的第三个 Tab。你可以点击右上角相机发布动态，AI 角色会根据人设给你点赞和评论。角色也会在后台随机触发主动发朋友圈。

生态记忆：系统会自动总结聊天记录和朋友圈互动，形成长期记忆。你可以在【我的】页面 -> 【记忆摘录】中查看和管理角色的记忆摘要。
5. 探索更多应用
日记：角色会根据和你的聊天内容，在后台偷偷写下私密日记。你可以在桌面的【日记】应用中偷看他们的内心世界。

查手机：桌面的【查手机】应用模拟了黑客提取数据的过程。你可以查看角色的搜索记录、购物订单、隐藏相册等深层隐私，内容由 AI 实时生成。

关系网：在【关系网】应用中，你可以为不同的角色建立社交关系。设定他们对彼此的看法后，甚至可以“窃听”他们之间的私聊记录。

全息论坛：一个模拟的公共社交平台。你可以发布帖子，角色们会来评论；角色们也会自己发帖互动。你可以在“论坛管理”中创建不同的板块并绑定特定的世界书。
6. 数据安全与备份
Moon小手机的所有数据（包括聊天记录、人设、图片等）均保存在你设备的本地浏览器缓存中（IndexedDB）。

强烈建议：定期在【设置】->【数据管理】中点击“导出备份数据”，将数据保存为 JSON 文件。更换设备或清理浏览器缓存前，请务必先导出备份，否则数据将永久丢失！
